# TechToday - Website Artikel Teknologi

Website ini adalah contoh sederhana halaman artikel bertema Teknologi/IT, dibuat menggunakan **HTML, CSS, dan JavaScript**.  
Fitur utama: artikel utama, daftar artikel (card), halaman detail artikel, video YouTube, desain modern responsif, dan tombol kembali ke atas.

---

## Fitur

- Menggunakan semantic HTML (header, nav, main, article, section, aside, footer)
- Card artikel dengan link ke halaman detail
- Halaman detail untuk setiap artikel
- Video YouTube dengan iframe
- Navigasi menu dan link sumber
- Desain modern, clean, dan responsif
- Efek hover, glassmorphism, dan tombol kembali ke atas
- Mobile friendly

---

## Cara Membuat & Menjalankan Project

### 1. **Clone atau Download Project**
Unduh semua file berikut dan simpan dalam satu folder:
- `index.html`
- `style.css`
- `script.js`
- `detail-ai.html`
- `detail-iot.html`
- `detail-blockchain.html`

### 2. **Struktur Folder**
```
/tugasAkhirWeb
│
├── index.html
├── style.css
├── script.js
├── detail-ai.html
├── detail-iot.html
└── detail-blockchain.html
```

### 3. **Penjelasan File**
- **index.html**  
  Halaman utama, berisi artikel utama, daftar card artikel, video, dan sumber.
- **style.css**  
  Semua styling modern dan responsif.
- **script.js**  
  Efek tombol kembali ke atas.
- **detail-ai.html**, **detail-iot.html**, **detail-blockchain.html**  
  Halaman detail untuk masing-masing artikel.

### 4. **Jalankan di Browser**
Cukup klik dua kali `index.html` atau buka dengan browser favorit Anda.

### 5. **Edit/Menambah Artikel**
- Untuk menambah card baru, duplikat struktur card di `<section class="card-list">` pada `index.html`.
- Untuk menambah halaman detail, duplikat salah satu file detail dan sesuaikan isinya.

### 6. **Kustomisasi**
- Ganti gambar, judul, dan isi artikel sesuai kebutuhan.
- Ubah warna atau font di `style.css` jika ingin tampilan berbeda.
